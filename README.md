# PDF Web Tool

A web-based PDF utility application built using Spring Boot.

## 🚀 Features
- Merge multiple PDFs
- Split PDF into separate pages
- Compress PDF files
- Resize PDF pages

## 🛠 Tech Stack
- Java
- Spring Boot
- Thymeleaf
- Maven

## 📌 Description
This application allows users to upload and process PDF files directly through a web interface.
